# Madoc documentation

The madoc platform is built upon some core technologies.

* IIIF
* Omeka
* Annotation studio
* Madoc vocabulary
* W3C Annotations (Elucidate)

It is used to enrich IIIF content with annotations and showcase IIIF content through Omeka websites.
